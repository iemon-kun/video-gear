# Video Gear

Chrome拡張で動画・音声の再生速度を手早く調整できます。

## Features
- タブ単位の再生速度設定
- トグルキーで100%と任意速度を切り替え
- スライダーとプリセットで素早く変速
- ほぼ全サイトの動画・音声に対応

## Setup
1. `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」から `chrome-extension` を選択

## Notes
- 設定はChrome同期で共有できます（拡張自体の同期は不可）
- 速度はサイトにより上限が異なる場合があります
